# Question
Approximately how many Apple I personal computers were created?

# Options
- [ ] A) 100
- [x] B) 200
- [ ] C) 500
- [ ] D) 1000

# Explanation
Approximately 200 Apple I computers were created by Steve Wozniak and Steve Jobs in 1976. The Apple I was one of the first personal computers to come with a fully assembled circuit board, though users still had to add a case, power supply, and keyboard.
- A) 100: This is too few
- C) 500: This overestimates the production
- D) 1000: This significantly overestimates the limited production run
