# Question
How many total bits are in a single precision floating point number?

# Options
- [ ] A) 16
- [x] B) 32
- [ ] C) 64
- [ ] D) 128

# Explanation
A single precision floating point number is 32 bits total, divided into: 1 sign bit, 8 exponent bits, and 23 significand bits. This is defined by the IEEE 754 standard.
- A) 16 bits would be half-precision floating point
- C) 64 bits is double precision floating point
- D) 128 bits is quadruple precision floating point
