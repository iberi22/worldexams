# Question
Which popular indie game was created using GameMaker Studio?

# Options
- [ ] A) Minecraft
- [x] B) Undertale
- [ ] C) Stardew Valley
- [ ] D) Terraria

# Explanation
Undertale, created by Toby Fox, was made using GameMaker Studio. This indie RPG became a massive success and demonstrated the capabilities of Mark Overmars' game engine.
- A) Minecraft was created in Java by Markus Persson
- C) Stardew Valley was created using C# and the MonoGame framework
- D) Terraria was created using C# and XNA framework
