# Question
What is a key feature of GameMaker that makes it beginner-friendly?

# Options
- [ ] A) Voice-controlled coding
- [x] B) Drag-and-drop interface
- [ ] C) Automatic AI generation
- [ ] D) Built-in 3D modeling

# Explanation
GameMaker, created by Mark Overmars, features a drag-and-drop interface that allows beginners to create games without extensive programming knowledge. Advanced users can also use GML (GameMaker Language) for more complex functionality.
- A) Voice-controlled coding is not a feature of GameMaker
- C) While AI features exist in modern tools, this isn't GameMaker's primary feature
- D) GameMaker is primarily focused on 2D games, not 3D modeling
