# Question
What type of routing protocol is OSPF (Open Shortest Path First)?

# Options
- [ ] A) Distance-vector protocol
- [x] B) Link-state protocol
- [ ] C) Path-vector protocol
- [ ] D) Hybrid protocol

# Explanation
OSPF is a link-state routing protocol. It maintains a complete topology map of the network and uses the Dijkstra algorithm to compute the shortest path.
- A) Distance-vector protocols (like RIP) share routing tables with neighbors
- C) Path-vector protocols (like BGP) are used for inter-domain routing
- D) Hybrid protocols (like EIGRP) combine features of distance-vector and link-state
