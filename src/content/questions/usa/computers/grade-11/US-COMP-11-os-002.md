# Question
What naming convention does Ubuntu use for its release names?

# Options
- [ ] A) Greek gods and planets
- [x] B) Alliterative adjectives and animals
- [ ] C) Famous scientists
- [ ] D) Mythical creatures only

# Explanation
Ubuntu uses alliterative adjectives and animal names for its releases (e.g., "Trusty Tahr," "Utopic Unicorn," "Wily Werewolf"). The first letter of both words in the name match, and each release increments alphabetically.
- A) Greek gods and planets are not part of Ubuntu's naming scheme
- C) Famous scientists are not used in Ubuntu release names
- D) While some releases use mythical creatures, not all animals are mythical
