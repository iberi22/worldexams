# Question
Dutch computer scientist Mark Overmars is known for creating which game development engine?

# Options
- [x] A) Game Maker
- [ ] B) Stencyl
- [ ] C) Construct
- [ ] D) Torque 2D

# Explanation
Mark Overmars created Game Maker (now known as GameMaker Studio), a popular game development platform that allows users to create games without extensive programming knowledge. It uses a drag-and-drop interface along with its own scripting language (GML).
- B) Stencyl was created by Jon Steinmetz
- C) Construct was developed by Scirra Ltd
- D) Torque 2D was developed by GarageGames
