# Question
Which processor did the Apple I use?

# Options
- [ ] A) Intel 8080
- [x] B) MOS 6502
- [ ] C) Motorola 68000
- [ ] D) Zilog Z80

# Explanation
The Apple I used the MOS 6502 processor running at 1 MHz. Steve Wozniak chose this processor because it was more affordable than the competing Intel 8080. The approximately 200 Apple I computers produced all used this chip.
- A) Intel 8080 was used in the Altair 8800 but was more expensive
- C) Motorola 68000 was used in later computers like the original Macintosh
- D) Zilog Z80 was used in computers like the TRS-80
