# Question
At which layer of the OSI model does OSPF (Open Shortest Path First) operate?

# Options
- [ ] A) Layer 2 (Data Link)
- [x] B) Layer 3 (Network)
- [ ] C) Layer 4 (Transport)
- [ ] D) Layer 7 (Application)

# Explanation
OSPF operates at Layer 3 (Network layer) of the OSI model, as it is responsible for routing IP packets across networks. It works directly with IP protocol.
- A) Layer 2 deals with MAC addresses and switches
- C) Layer 4 deals with TCP/UDP and port numbers
- D) Layer 7 deals with application-level protocols like HTTP
