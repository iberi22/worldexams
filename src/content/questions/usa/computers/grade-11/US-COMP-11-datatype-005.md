# Question
Which component of a floating point number determines whether it is positive or negative?

# Options
- [x] A) Sign bit
- [ ] B) Exponent
- [ ] C) Significand (mantissa)
- [ ] D) Base

# Explanation
The sign bit (1 bit) determines whether a floating point number is positive (0) or negative (1). In single precision format, this is the first bit, followed by 8 exponent bits and 23 significand bits.
- B) The exponent determines the magnitude/scale, not the sign
- C) The significand (23 bits in single precision) represents the precision, not sign
- D) The base is implicit (base-2) and not stored
