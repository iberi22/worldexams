# Question
In what year was GameMaker first released by Mark Overmars?

# Options
- [ ] A) 1995
- [x] B) 1999
- [ ] C) 2003
- [ ] D) 2007

# Explanation
GameMaker was first released by Mark Overmars in 1999. It started as a simple tool for creating games and evolved into GameMaker Studio, a professional-grade game development platform used by indie developers worldwide.
- A) 1995: This is too early; Overmars was still developing the concept
- C) 2003: This is when later versions were gaining popularity
- D) 2007: This is closer to when GameMaker Studio branding emerged
