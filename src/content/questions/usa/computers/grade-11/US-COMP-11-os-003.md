# Question
What does "LTS" stand for in Ubuntu releases like "Trusty Tahr 14.04 LTS"?

# Options
- [ ] A) Latest Technical Support
- [x] B) Long Term Support
- [ ] C) Limited Time Stable
- [ ] D) Linux Terminal System

# Explanation
LTS stands for "Long Term Support," which means the release receives security updates and support for 5 years (or longer), unlike regular releases which are supported for 9 months. Trusty Tahr 14.04 LTS was one such long-term support release.
- A) Latest Technical Support is not what LTS means
- C) Limited Time Stable contradicts the purpose of LTS
- D) Linux Terminal System is not a real term
