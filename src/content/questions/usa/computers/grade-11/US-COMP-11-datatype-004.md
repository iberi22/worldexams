# Question
What is the standard that defines the format for single precision floating point numbers?

# Options
- [ ] A) ASCII
- [ ] B) Unicode
- [x] C) IEEE 754
- [ ] D) ISO 8859

# Explanation
IEEE 754 is the standard that defines floating point arithmetic, including the single precision format (32-bit with 23-bit significand, 8-bit exponent, and 1 sign bit). This standard is used by virtually all modern computers.
- A) ASCII is a character encoding standard
- B) Unicode is a text encoding standard
- D) ISO 8859 is a character encoding standard
