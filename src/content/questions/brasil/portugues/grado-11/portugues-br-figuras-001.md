# Pergunta
Na frase "Aquele político é uma raposa", qual figura de linguagem está presente?

# Opções
- [ ] A) Metonímia
- [x] B) Metáfora
- [ ] C) Hipérbole
- [ ] D) Eufemismo

# Explicação
A metáfora é uma comparação implícita sem conectivo. Ao chamar o político de "raposa", atribui-se a ele características como astúcia e esperteza, típicas do animal.
- A) Metonímia: substituição por relação de contiguidade
- C) Hipérbole: exagero intencional
- D) Eufemismo: suavização de expressões
