# Pergunta

Na cidade de São Paulo, um laboratório de física está estudando o comportamento de materiais em temperaturas extremas. Os alunos precisam encontrar o ponto onde um termômetro Celsius marca o mesmo valor que um termômetro Fahrenheit. Qual é essa temperatura?

# Opções

- [ ] A) 0°
- [x] B) -40°
- [ ] C) 100°
- [ ] D) -273°

# Explicação

Resolvendo a equação F = C: x = (9/5)x + 32, onde x é a temperatura procurada. Multiplicando por 5: 5x = 9x + 160. Então: -4x = 160, logo x = -40°. Esta é a única temperatura onde ambas as escalas coincidem.

- A) Zero é o ponto de congelamento da água em Celsius, mas 32°F
- C) 100°C é a ebulição da água, mas equivale a 212°F
- D) -273°C é o zero absoluto, mas equivale a -459°F
