# Pergunta

Uma pesquisadora do INPE (Instituto Nacional de Pesquisas Espaciais) está calibrando instrumentos que funcionam com diferentes escalas de temperatura. Ela precisa identificar o ponto de intersecção das escalas Celsius e Fahrenheit para garantir a precisão das medições. Qual valor ela deve usar?

# Opções

- [ ] A) 0°
- [ ] B) -32°
- [x] C) -40°
- [ ] D) 273°

# Explicação

O ponto de intersecção das escalas Celsius e Fahrenheit é -40°. Este é um valor importante na calibração de instrumentos que trabalham com ambas as escalas. Algebricamente, resolvendo C = (5/9)(F - 32) quando C = F, obtemos -40°.

- A) 0° é referência para o gelo, mas em escalas diferentes
- B) -32° não é ponto de intersecção
- D) 273 está relacionado à escala Kelvin, não à intersecção C/F
