# Pergunta

Um cientista brasileiro está conduzindo experimentos em temperaturas extremamente baixas. Ao converter -40°C para a escala Fahrenheit, qual valor ele obterá?

# Opções

- [ ] A) -32°F
- [ ] B) -72°F
- [x] C) -40°F
- [ ] D) -104°F

# Explicação

Quando uma temperatura é -40°C, seu equivalente em Fahrenheit também é -40°F. Este é o único ponto onde as duas escalas coincidem. Usando F = (9/5)C + 32, temos: F = (9/5)(-40) + 32 = -72 + 32 = -40°F.

- A) Esqueceu de aplicar o fator 9/5
- B) Esqueceu de adicionar 32
- D) Multiplicou erroneamente por 9/5 após adicionar 32
