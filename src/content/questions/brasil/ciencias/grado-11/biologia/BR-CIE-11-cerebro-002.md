# Pergunta

Em uma aula de neurociência na UNIFESP em São Paulo, o professor explica que uma estrutura cerebral foi nomeada devido à sua aparência que lembra um animal marinho. Essa estrutura, fundamental para a memória, é o:

# Opções

- [ ] A) Corpo caloso
- [x] B) Hipocampo
- [ ] C) Hipotálamo
- [ ] D) Córtex cerebral

# Explicação

O hipocampo tem formato semelhante a um cavalo-marinho, daí seu nome (do grego "hippos" = cavalo + "kampos" = criatura marinha). Localizado no lobo temporal, é essencial para consolidação de memórias de curto para longo prazo.

- A) Corpo caloso conecta os hemisférios, não tem forma de animal
- C) Hipotálamo controla homeostase, nome vem de "abaixo do tálamo"
- D) Córtex cerebral é a camada externa do cérebro
