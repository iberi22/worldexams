# Pergunta

Pesquisadores brasileiros estudando o mal de Alzheimer focam em uma estrutura cerebral cuja forma lembra um cavalo-marinho e que é crítica para a memória. Esta estrutura é:

# Opções

- [ ] A) Amígdala cerebral
- [ ] B) Cerebelo
- [x] C) Hipocampo
- [ ] D) Ponte

# Explicação

O hipocampo é a resposta correta. Seu nome deriva do grego "hippokampos" (cavalo-marinho) devido à sua forma característica. No Alzheimer, esta é uma das primeiras regiões afetadas, explicando os problemas de memória iniciais da doença.

- A) Amígdala tem forma de amêndoa, não cavalo-marinho
- B) Cerebelo coordena movimentos, não processa memórias
- D) Ponte conecta partes do cérebro, não tem relação com memória
