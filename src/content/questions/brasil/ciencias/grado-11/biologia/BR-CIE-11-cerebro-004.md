# Pergunta

Na anatomia cerebral, qual estrutura deve seu nome à semelhança com a palavra grega para cavalo-marinho e é responsável pela formação de novas memórias?

# Opções

- [ ] A) Neocórtex
- [ ] B) Bulbo
- [ ] C) Tálamo
- [x] D) Hipocampo

# Explicação

O hipocampo (do grego "hippokampos" = cavalo-marinho) é uma estrutura em forma de C localizada no lobo temporal medial. É essencial para transformar memórias de curto prazo em memórias de longo prazo, processo chamado consolidação.

- A) Neocórtex é a camada mais recente evolutivamente
- B) Bulbo controla funções vitais como respiração
- C) Tálamo é o "portão" sensorial do cérebro
