# Pergunta

Um laboratório do CPRM (Serviço Geológico do Brasil) está testando a dureza de minerais extraídos da Amazônia. Para comparação, eles usam a escala de Mohs. Qual mineral serve como referência para o grau 1 (mais baixo)?

# Opções

- [ ] A) Bauxita
- [ ] B) Hematita
- [x] C) Talco
- [ ] D) Caulim

# Explicação

O talco é o mineral padrão para dureza 1 na escala de Mohs. Quimicamente é um filossilicato de magnésio hidratado. Sua extrema maciez o torna útil em várias aplicações industriais, desde papel até tintas. Brasil é um grande produtor mundial de talco.

- A) Bauxita é minério de alumínio, dureza 1-3
- B) Hematita é minério de ferro, dureza 5-6
- D) Caulim é argila, dureza variável
