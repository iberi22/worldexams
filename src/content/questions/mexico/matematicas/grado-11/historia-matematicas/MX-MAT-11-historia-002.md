# Pregunta

Entre las múltiples contribuciones de Leonhard Euler, ¿cuál de las siguientes NO es auténtica?

# Opciones

- [ ] A) La notación f(x) para funciones
- [ ] B) El símbolo e para la base de logaritmos naturales
- [x] C) El algoritmo de multiplicación de matrices de Strassen
- [ ] D) La función zeta de Riemann (estudiada primero por Euler)

# Explicación

El algoritmo de Strassen para multiplicación de matrices fue desarrollado por Volker Strassen en 1969, casi 200 años después de Euler. Las demás son contribuciones genuinas de Euler.
- A) Euler introdujo la notación moderna de funciones
- B) Euler popularizó el uso de e
- D) Euler estudió la función zeta antes que Riemann
