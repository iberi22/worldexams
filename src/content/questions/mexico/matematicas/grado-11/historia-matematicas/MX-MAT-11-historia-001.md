# Pregunta

¿Cuál de las siguientes contribuciones NO fue desarrollada por el matemático Leonhard Euler?

# Opciones

- [ ] A) Un método para resolver ecuaciones diferenciales de primer orden
- [ ] B) Una identidad que relaciona e, π e i
- [ ] C) Una fórmula que relaciona vértices, aristas y caras en grafos
- [x] D) Una mejora a la Transformada Rápida de Fourier

# Explicación

La Transformada Rápida de Fourier (FFT) fue desarrollada en el siglo XX, mucho después de Euler (1707-1783). Euler sí desarrolló los otros tres conceptos mencionados.
- A) Euler desarrolló métodos para EDOs de primer orden
- B) La identidad de Euler: e^(iπ) + 1 = 0
- C) La fórmula de Euler para poliedros: V - A + C = 2
