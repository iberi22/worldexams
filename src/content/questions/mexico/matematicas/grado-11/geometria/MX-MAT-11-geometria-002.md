# Pregunta
¿Cuál es el área de un círculo con un diámetro de 14 metros si π = 3.1416?

# Opciones
- [ ] A) 87.96 metros cuadrados
- [x] B) 153.94 metros cuadrados
- [ ] C) 615.75 metros cuadrados
- [ ] D) 43.98 metros cuadrados

# Explicación
El área de un círculo se calcula con la fórmula A = πr², donde r es el radio.
- Radio: r = 14 ÷ 2 = 7 metros
- Área: A = 3.1416 × 7² = 3.1416 × 49 = 153.94 metros cuadrados

Errores comunes:
- A) 87.96: Error al usar un radio de 5.3 metros aproximadamente
- C) 615.75: Error al usar el diámetro como radio (14² × 3.1416)
- D) 43.98: Error al calcular solo πr sin elevar al cuadrado
