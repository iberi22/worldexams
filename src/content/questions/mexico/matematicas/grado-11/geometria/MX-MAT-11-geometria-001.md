# Pregunta
¿Cuál es el área de un círculo con un diámetro de 20 pulgadas si π = 3.1415?

# Opciones
- [x] A) 314.15 pulgadas cuadradas
- [ ] B) 380.1215 pulgadas cuadradas
- [ ] C) 3141.5 pulgadas cuadradas
- [ ] D) 1256.6 pulgadas cuadradas

# Explicación
El área de un círculo se calcula con la fórmula A = πr², donde r es el radio.
- Primero, calculamos el radio: r = diámetro ÷ 2 = 20 ÷ 2 = 10 pulgadas
- Luego, aplicamos la fórmula: A = 3.1415 × 10² = 3.1415 × 100 = 314.15 pulgadas cuadradas

Errores comunes:
- B) 380.1215: Error al multiplicar incorrectamente
- C) 3141.5: Error al no dividir el diámetro por 2 (usando el diámetro como radio)
- D) 1256.6: Resultado de usar π = 3.1415 y radio = 20 (confusión con otro cálculo)
