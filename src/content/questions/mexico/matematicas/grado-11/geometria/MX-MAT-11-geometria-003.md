# Pregunta
¿Cuál es el área de un círculo con un radio de 8 centímetros si π = 3.1416?

# Opciones
- [ ] A) 25.13 centímetros cuadrados
- [ ] B) 50.27 centímetros cuadrados
- [x] C) 201.06 centímetros cuadrados
- [ ] D) 804.25 centímetros cuadrados

# Explicación
El área de un círculo se calcula con la fórmula A = πr².
- Radio dado: r = 8 centímetros
- Área: A = 3.1416 × 8² = 3.1416 × 64 = 201.06 centímetros cuadrados

Errores comunes:
- A) 25.13: Resultado de usar π × r (sin elevar al cuadrado)
- B) 50.27: Resultado de calcular la circunferencia (2πr) en lugar del área
- D) 804.25: Error al usar el diámetro (16) como radio
