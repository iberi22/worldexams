# Pregunta
Si la circunferencia de un círculo es 31.416 metros (usando π = 3.1416), ¿cuál es su área?

# Opciones
- [ ] A) 31.42 metros cuadrados
- [ ] B) 50.27 metros cuadrados
- [x] C) 78.54 metros cuadrados
- [ ] D) 157.08 metros cuadrados

# Explicación
Primero debemos encontrar el radio usando la fórmula de la circunferencia C = 2πr:
- C = 31.416 metros
- 31.416 = 2 × 3.1416 × r
- 31.416 = 6.2832 × r
- r = 31.416 ÷ 6.2832 = 5 metros

Ahora calculamos el área:
- A = πr² = 3.1416 × 5² = 3.1416 × 25 = 78.54 metros cuadrados

Errores comunes:
- A) 31.42: Confusión entre circunferencia y área
- B) 50.27: Cálculo incorrecto del radio
- D) 157.08: Error al duplicar el resultado correcto
