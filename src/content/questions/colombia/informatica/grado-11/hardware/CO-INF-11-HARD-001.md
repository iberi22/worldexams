# Pregunta
¿Cuál es el procesador principal (CPU) utilizado en la consola Sega Mega Drive, conocida como Sega Genesis en América?

# Opciones
- [x] A) Motorola 68000
- [ ] B) Zilog Z80
- [ ] C) Yamaha YM2612
- [ ] D) Intel 8088

# Explicación
El Sega Mega Drive/Genesis utiliza el Motorola 68000 como CPU principal, funcionando a 7.6 MHz. El Zilog Z80 es usado como co-procesador para compatibilidad con Sega Master System. El Yamaha YM2612 es el chip de sonido, no un procesador principal. El Intel 8088 fue usado en la IBM PC original.
