# Pregunta
En el proceso de digitalización de audio para CDs, se utiliza una técnica llamada PCM (Pulse Code Modulation). ¿A qué frecuencia se muestrea el audio en este formato?

# Opciones
- [ ] A) 48.0 kHz (estándar para video profesional)
- [ ] B) 96.0 kHz (audio de alta resolución)
- [x] C) 44.1 kHz (estándar CD)
- [ ] D) 22.05 kHz (calidad de voz)

# Explicación
El formato CD utiliza PCM con 44.1 kHz de frecuencia de muestreo. Esta frecuencia fue elegida porque es ligeramente superior al doble del rango audible humano máximo (20 kHz), cumpliendo con el teorema de Nyquist-Shannon. Aunque 48 kHz se usa en video y 96 kHz en audio de alta resolución, el estándar CD es 44.1 kHz.
