# Pregunta

¿Cómo denomina el Sistema Internacional de Cantidades a 1024 bytes?

# Opciones

- [ ] A) Kilobyte
- [ ] B) Kylobyte
- [x] C) Kibibyte
- [ ] D) Kelobyte

# Explicación

El Sistema Internacional de Cantidades (IEC) utiliza el término "Kibibyte" (KiB) para referirse a 1024 bytes, diferenciándolo del "Kilobyte" (KB) que en el sistema decimal equivale a 1000 bytes.

- A es incorrecto: Kilobyte en el sistema decimal equivale a 1000 bytes, no 1024.
- B y D son incorrectos: Son nombres inventados que no existen en ningún sistema de unidades.
- C es correcto: Kibibyte es la unidad oficial del IEC para 1024 bytes (2^10).

**Competencia:** Conocimiento de unidades de medida en informática y sistemas de numeración binario.
