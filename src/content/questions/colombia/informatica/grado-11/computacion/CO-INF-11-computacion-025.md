# Pregunta

El dominio de internet .fm es el dominio de nivel superior de código de país para qué nación insular del Océano Pacífico?

# Opciones

- [ ] A) Fiji
- [x] B) Micronesia
- [ ] C) Islas Marshall
- [ ] D) Tuvalu

# Explicación

.fm es el dominio de nivel superior de código de país (ccTLD) asignado a los Estados Federados de Micronesia. Es popular entre estaciones de radio y servicios de streaming por su asociación con "FM" (Frecuencia Modulada).

- A es incorrecto: El dominio de Fiji es .fj
- B es correcto: .fm corresponde a Micronesia (Federated States of Micronesia).
- C es incorrecto: Las Islas Marshall usan el dominio .mh
- D es incorrecto: Tuvalu usa el dominio .tv (también popular comercialmente).

**Competencia:** Conocimiento de dominios de internet y geografía digital.
