# Pregunta

Una estación de radio en línea quiere registrar un dominio que refleje su tipo de servicio. ¿Qué dominio de nivel superior sería más apropiado por su asociación con radio?

# Opciones

- [ ] A) .am (Armenia)
- [x] B) .fm (Micronesia)
- [ ] C) .mu (Mauricio)
- [ ] D) .so (Somalia)

# Explicación

El dominio .fm de Micronesia es ideal para servicios de radio porque "FM" es universalmente reconocido como Frecuencia Modulada, el sistema estándar de transmisión de radio. Aunque también existe .am (Armenia) que se asocia con Amplitud Modulada, .fm es más popular.

- A es incorrecto aunque podría funcionar: .am pertenece a Armenia, pero AM (Amplitud Modulada) es menos popular que FM.
- B es correcto: .fm es perfecto por su asociación con FM (Frecuencia Modulada).
- C es incorrecto: .mu pertenece a Mauricio y no tiene asociación con radio.
- D es incorrecto: .so pertenece a Somalia y no tiene relación con servicios de audio.

**Competencia:** Aplicación de conocimientos de dominios para casos de uso específicos.
