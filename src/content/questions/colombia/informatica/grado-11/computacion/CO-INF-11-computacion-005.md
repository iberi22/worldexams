# Pregunta

En un sistema operativo, una aplicación reporta usar 4 KiB de memoria. ¿A cuántos bytes exactamente equivale esto según el estándar IEC?

# Opciones

- [ ] A) 4000 bytes
- [x] B) 4096 bytes
- [ ] C) 4024 bytes
- [ ] D) 4100 bytes

# Explicación

4 KiB (Kibibytes) = 4 × 1024 = 4096 bytes, utilizando el sistema binario del IEC.

- A es incorrecto: 4000 bytes sería 4 KB (Kilobytes en sistema decimal).
- B es correcto: 4 × 1024 = 4096 bytes exactos.
- C es incorrecto: Es una cantidad arbitraria sin base matemática.
- D es incorrecto: No corresponde a ningún cálculo estándar.

**Competencia:** Cálculo y conversión de unidades de memoria.
