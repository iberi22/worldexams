# Pregunta

En el Sistema Internacional de Cantidades, ¿cuál es la unidad correcta para representar 1024 bytes de información?

# Opciones

- [ ] A) Megabyte
- [x] B) Kibibyte
- [ ] C) Kilobyte
- [ ] D) Gigabyte

# Explicación

El IEC (International Electrotechnical Commission) estableció el Kibibyte como la unidad para representar 1024 bytes, usando el prefijo binario "kibi" (Ki).

- A es incorrecto: Un Megabyte es mucho mayor (1024 KiB o 1000 KB).
- B es correcto: Kibibyte es la unidad oficial para 1024 bytes.
- C es incorrecto: En el sistema decimal, Kilobyte representa 1000 bytes.
- D es incorrecto: Un Gigabyte es mucho mayor (1024 MiB o 1000 MB).
