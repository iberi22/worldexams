# Pregunta
¿Cuál es el nombre de la capa 4 del modelo OSI (Open Systems Interconnection)?

# Opciones
- [x] A) Transporte
- [ ] B) Sesión
- [ ] C) Enlace de Datos
- [ ] D) Red

# Explicación
La capa 4 del modelo OSI es la capa de Transporte. Esta capa es responsable de la entrega confiable de datos entre sistemas finales, incluyendo control de flujo y corrección de errores. La capa de Sesión es la 5, Enlace de Datos es la 2, y Red es la 3.
