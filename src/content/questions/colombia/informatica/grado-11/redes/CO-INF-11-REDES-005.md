# Pregunta
Al enviar datos por red, diferentes capas del modelo OSI añaden información. ¿Qué capa trabaja con "segmentos" y números de puerto?

# Opciones
- [ ] A) Capa de Red (trabaja con paquetes)
- [x] B) Capa de Transporte (trabaja con segmentos)
- [ ] C) Capa de Enlace (trabaja con tramas)
- [ ] D) Capa Física (trabaja con bits)

# Explicación
La Capa de Transporte (Layer 4) trabaja con segmentos y utiliza números de puerto para identificar aplicaciones específicas. Por ejemplo, HTTP usa el puerto 80, HTTPS el 443. Los segmentos son la unidad de datos de esta capa, mientras que paquetes, tramas y bits corresponden a capas 3, 2 y 1 respectivamente.
