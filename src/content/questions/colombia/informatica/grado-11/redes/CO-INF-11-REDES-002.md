# Pregunta
En el modelo de referencia OSI de 7 capas, la capa que proporciona servicios de transferencia de datos de extremo a extremo y control de flujo es:

# Opciones
- [ ] A) Capa de Red (Layer 3)
- [x] B) Capa de Transporte (Layer 4)
- [ ] C) Capa de Sesión (Layer 5)
- [ ] D) Capa de Aplicación (Layer 7)

# Explicación
La Capa de Transporte (Layer 4) es responsable de la comunicación de extremo a extremo entre hosts. Implementa protocolos como TCP (confiable) y UDP (no confiable). La capa de Red maneja direccionamiento y enrutamiento, la de Sesión maneja diálogos, y la de Aplicación interactúa con el usuario.
