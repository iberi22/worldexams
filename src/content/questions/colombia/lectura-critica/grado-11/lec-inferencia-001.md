# Pregunta
Lee el siguiente fragmento:

*"Juan llegó a casa empapado, sacudiéndose el agua del cabello mientras buscaba una toalla."*

¿Qué se puede inferir del texto?

# Opciones
- [ ] A) Juan se bañó recientemente
- [x] B) Estaba lloviendo afuera
- [ ] C) Juan practicó natación
- [ ] D) Hubo una inundación

# Explicación
La descripción de llegar empapado y sacudirse el agua sugiere que probablemente estaba lloviendo cuando Juan llegó.
