# Pregunta
**Texto:**
"Para preparar un buen café, primero se debe moler el grano justo antes de la preparación. Luego, se calienta el agua a unos 90°C, sin dejar que hierva. Finalmente, se vierte el agua sobre el café molido en movimientos circulares."

El texto anterior es de tipo:

# Opciones
- [ ] A) Argumentativo.
- [ ] B) Narrativo.
- [x] C) Instructivo o Expositivo.
- [ ] D) Lírico.

# Explicación Pedagógica

**¿Por qué C es correcta?**
El texto es instructivo porque presenta una secuencia de pasos ordenados (primero, luego, finalmente) para realizar una tarea específica: preparar café. Los textos instructivos tienen como propósito guiar al lector para lograr un objetivo práctico.

**¿Por qué las otras opciones son incorrectas?**
- **A) Argumentativo:** Un texto argumentativo defiende una posición con razones y evidencias. Este texto no defiende ninguna opinión.
- **B) Narrativo:** Un texto narrativo cuenta una historia con personajes y eventos. No hay personajes ni trama aquí.
- **D) Lírico:** Un texto lírico expresa sentimientos y emociones de forma poética. Este texto es práctico, no emocional.
