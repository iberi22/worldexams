# Pregunta
¿Cuántos cromosomas tiene una célula humana somática normal?

# Opciones
- [ ] A) 23
- [x] B) 46
- [ ] C) 48
- [ ] D) 92

# Explicación
Las células somáticas humanas tienen 46 cromosomas (23 pares), mientras que las células sexuales (gametos) tienen 23.
