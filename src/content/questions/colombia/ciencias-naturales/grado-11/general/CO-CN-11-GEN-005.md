# Pregunta
Un estudiante debe resolver: "Si aplicamos la fórmula F = (9/5)C + 32 y queremos que F = C, ¿cuál es el valor de temperatura que satisface esta condición?"

# Opciones
- [x] A) -40
- [ ] B) 0
- [ ] C) -273
- [ ] D) 32

# Explicación
Para encontrar dónde F = C, debemos resolver:
C = (9/5)C + 32
C - (9/5)C = 32
(-4/5)C = 32
C = -40

Por lo tanto, -40 es la única temperatura donde Celsius y Fahrenheit tienen el mismo valor numérico.

Las opciones incorrectas:
- B) En 0°C, Fahrenheit es 32°F (no son iguales)
- C) -273 es cercano al cero absoluto pero no satisface F = C
- D) 32 es la conversión de 0°C a Fahrenheit, no un punto de igualdad
