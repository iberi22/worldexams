# Pregunta
El enlace iónico se forma principalmente entre:

# Opciones
- [ ] A) Dos metales
- [ ] B) Dos no metales
- [x] C) Un metal y un no metal
- [ ] D) Gases nobles

# Explicación Pedagógica

**¿Por qué C es correcta?**
El enlace iónico se forma cuando hay una transferencia de electrones de un átomo a otro. Los metales tienden a perder electrones (formando cationes) y los no metales tienden a ganarlos (formando aniones). La atracción electroestática entre estos iones opuestos forma el enlace iónico. Ejemplo: NaCl (sal de mesa).

**¿Por qué las otras opciones son incorrectas?**
- **A) Dos metales:** Entre metales se forma el enlace metálico, con electrones deslocalizados compartidos.
- **B) Dos no metales:** Entre no metales se forman enlaces covalentes, compartiendo electrones.
- **D) Gases nobles:** Los gases nobles son muy estables y raramente forman enlaces químicos.
