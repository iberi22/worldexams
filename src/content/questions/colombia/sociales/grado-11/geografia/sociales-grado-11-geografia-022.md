# Pregunta

¿Qué ciudad colombiana fue fundada originalmente con el nombre de "Santiago de Cali"?

# Opciones

- [ ] A) Santiago de Chile
- [x] B) Cali
- [ ] C) Calarcá
- [ ] D) Cartago

# Explicación

Cali fue fundada el 25 de julio de 1536 por Sebastián de Belalcázar con el nombre de "Santiago de Cali". El nombre "Santiago" era común para honrar al apóstol Santiago, patrón de España. Con el tiempo, el nombre se simplificó a Cali.
