# Pregunta

¿Qué método innovador de comunicación secreta utilizaba el Culper Ring para transmitir información a George Washington?

# Opciones

- [ ] A) Mensajes transmitidos únicamente de forma oral
- [ ] B) Señales de humo desde las montañas
- [x] C) Tinta invisible y un código numérico cifrado
- [ ] D) Palomas mensajeras entrenadas

# Explicación

La respuesta correcta es **C) Tinta invisible y un código numérico cifrado.**

El Culper Ring utilizaba técnicas sofisticadas para su época: tinta invisible (desarrollada por James Jay) que solo aparecía al aplicarse un reactivo específico, y el "Culper Code Book", un sistema de cifrado numérico donde cada número representaba una palabra o nombre. Por ejemplo, "711" significaba George Washington y "745" era Inglaterra. Esta combinación hacía los mensajes prácticamente indescifrables para los británicos.

**Errores comunes:**
- A: Los mensajes escritos eran esenciales para documentar información detallada.
- B: Nueva York era territorio llano y urbano, inadecuado para señales de humo.
- D: Aunque se usaron palomas mensajeras en otros conflictos, el Culper Ring dependía de correos humanos.

**Competencia:** Comprensión de tecnologías de comunicación y criptografía histórica.
