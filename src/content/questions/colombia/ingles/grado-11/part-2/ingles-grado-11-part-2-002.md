# Pregunta
Match the definition with the correct word.
"This is a place where you can buy medicine."

# Opciones
- [ ] A) Library
- [ ] B) Bakery
- [x] C) Drugstore / Pharmacy
- [ ] D) Butcher shop

# Explicación
- **Drugstore/Pharmacy (Farmacia):** Lugar para comprar medicinas.
- Library (Biblioteca): Lugar para leer/prestar libros.
- Bakery (Panadería): Lugar para comprar pan.
- Butcher shop (Carnicería): Lugar para comprar carne.

**Competencia:** Conocimiento léxico.
