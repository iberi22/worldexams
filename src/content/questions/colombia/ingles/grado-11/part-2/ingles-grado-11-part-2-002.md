# Pregunta
Match the definition with the correct word.
"This is a place where you can buy medicine."

# Opciones
- [ ] A) Library
- [ ] B) Bakery
- [x] C) Drugstore / Pharmacy
- [ ] D) Butcher shop

# Explicación Pedagógica

**¿Por qué C es correcta?**
"Drugstore" o "Pharmacy" (farmacia) es el establecimiento comercial especializado en la venta de medicamentos, productos de higiene y salud.

**¿Por qué las otras opciones son incorrectas?**
- **A) Library (Biblioteca):** Es un lugar para leer, estudiar o prestar libros, no para comprar medicina.
- **B) Bakery (Panadería):** Es donde se elabora y vende pan y productos horneados.
- **D) Butcher shop (Carnicería):** Es donde se vende carne.
