# Pregunta
Select the correct form: "If I ___ rich, I would travel the world."

# Opciones
- [ ] A) am
- [x] B) were
- [ ] C) was
- [ ] D) be

# Explicación
This is a second conditional (hypothetical situation). We use "were" for all subjects in the if-clause.
