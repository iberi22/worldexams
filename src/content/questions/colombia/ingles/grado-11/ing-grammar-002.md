# Pregunta
Select the correct form: "If I ___ rich, I would travel the world."

# Opciones
- [ ] A) am
- [x] B) were
- [ ] C) was
- [ ] D) be

# Explicación Pedagógica

**¿Por qué B es correcta?**
Esta oración es un ejemplo del segundo condicional (Second Conditional), que se usa para expresar situaciones hipotéticas o irreales en el presente o futuro. La estructura es: If + past simple, would + base verb. En el segundo condicional, se usa "were" para todos los sujetos (I, you, he, she, it, we, they), especialmente en contextos formales. "If I were rich" es la forma correcta y estándar.

**¿Por qué las otras opciones son incorrectas?**
- **A) am:** "Am" es presente simple. El segundo condicional requiere pasado simple en la cláusula if.
- **C) was:** Aunque "was" se usa informalmente, "were" es la forma gramaticalmente correcta para el subjuntivo en condicionales irreales.
- **D) be:** El infinitivo "be" no es correcto después de "If I". Se necesita un verbo conjugado.
