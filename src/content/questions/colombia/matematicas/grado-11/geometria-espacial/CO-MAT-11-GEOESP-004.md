# Pregunta

¿Cuál de las siguientes fórmulas NO corresponde a una esfera?

# Opciones

- [ ] A) Volumen: $\frac{4}{3}\pi r^3$
- [ ] B) Área superficial: $4\pi r^2$
- [x] C) Volumen: $\pi r^2 h$
- [ ] D) Diámetro: $2r$

# Explicación

La opción C ($V = \pi r^2 h$) es la fórmula del volumen de un cilindro, no de una esfera.
- Las opciones A y B son fórmulas correctas para esfera.
- La opción D es la relación correcta entre radio y diámetro.
