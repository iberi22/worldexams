# Pregunta

¿Cuál es la fórmula correcta para calcular el volumen de una esfera?

# Opciones

- [ ] A) $4\pi r^2$
- [x] B) $\frac{4}{3}\pi r^3$
- [ ] C) $\frac{1}{3}\pi h r^2$
- [ ] D) $\pi r^4$

# Explicación

La fórmula del volumen de una esfera es $V = \frac{4}{3}\pi r^3$, donde $r$ es el radio.
- **Opción A** ($4\pi r^2$) es la fórmula del área superficial de una esfera.
- **Opción C** ($\frac{1}{3}\pi h r^2$) es la fórmula del volumen de un cono.
- **Opción D** ($\pi r^4$) no corresponde a ninguna fórmula geométrica estándar.
