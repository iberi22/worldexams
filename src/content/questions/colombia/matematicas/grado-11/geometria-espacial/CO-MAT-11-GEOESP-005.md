# Pregunta

En Bogotá, una empresa de tanques de agua fabrica tanques esféricos. Si el área superficial de un tanque esférico es $A = 4\pi r^2$, ¿cuál es la relación entre el área superficial y el volumen?

# Opciones

- [ ] A) $A = 2V$
- [ ] B) $A = 3V/r$
- [x] C) $A = 3V/r$
- [ ] D) $A = V/r$

# Explicación

Dado que $A = 4\pi r^2$ y $V = \frac{4}{3}\pi r^3$, podemos despejar:
$V = \frac{4}{3}\pi r^3$ → $\frac{3V}{r} = 4\pi r^2 = A$

Por lo tanto, $A = \frac{3V}{r}$.
