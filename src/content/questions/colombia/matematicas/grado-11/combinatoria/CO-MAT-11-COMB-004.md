# Pregunta

¿Cuál es el orden de magnitud correcto para expresar $10^{20}$ en palabras?

# Opciones

- [ ] A) Mil billones
- [ ] B) Cien trillones
- [x] C) Cien mil trillones
- [ ] D) Un trillón

# Explicación

En notación internacional:
- $10^{12}$ = Un billón (trillion en inglés)
- $10^{18}$ = Un trillón (quintillion en inglés)
- $10^{20}$ = Cien mil trillones

Las máquinas de cifrado complejas pueden tener este orden de combinaciones posibles.
