# Pregunta
Simplifica la expresión: $3(x + 2) - 2(x - 1)$

# Opciones
- [ ] A) $x + 4$
- [ ] B) $x + 5$
- [x] C) $x + 8$
- [ ] D) $5x + 4$

# Explicación Pedagógica

**¿Por qué C es correcta?**
Para simplificar la expresión, primero aplicamos la propiedad distributiva a cada término. Para $3(x + 2)$, multiplicamos 3 por cada término dentro del paréntesis: $3 \cdot x + 3 \cdot 2 = 3x + 6$. Para $-2(x - 1)$, multiplicamos -2 por cada término: $-2 \cdot x - 2 \cdot (-1) = -2x + 2$. Luego, combinamos los resultados: $(3x + 6) + (-2x + 2)$. Agrupamos los términos semejantes: $(3x - 2x) + (6 + 2)$, lo que nos da $x + 8$.

**¿Por qué las otras opciones son incorrectas?**
- **A) $x + 4$:** Este error ocurre si el signo negativo no se distribuye correctamente al segundo término del segundo paréntesis, resultando en $3x + 6 - 2x - 2 = x + 4$.
- **B) $x + 5$:** Este error puede surgir de un error de cálculo al sumar los términos constantes (6 y -1, por ejemplo), o al distribuir incorrectamente.
- **D) $5x + 4$:** Este resultado se obtiene si se suman los coeficientes de x (3 y 2) en lugar de restarlos ($3x+2x=5x$) y se distribuye incorrectamente el signo negativo ($6-2=4$).
