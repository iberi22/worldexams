# Pregunta
Simplifica la expresión: $3(x + 2) - 2(x - 1)$

# Opciones
- [ ] A) $x + 4$
- [ ] B) $x + 5$
- [x] C) $x + 8$
- [ ] D) $5x + 4$

# Explicación
$3(x + 2) - 2(x - 1) = 3x + 6 - 2x + 2 = x + 8$
