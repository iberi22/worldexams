# Pregunta
¿Cuál es el valor de la expresión $\frac{x^2 - 9}{x - 3}$ cuando $x \neq 3$?

# Opciones
- [ ] A) $x - 3$
- [x] B) $x + 3$
- [ ] C) $x^2 - 3$
- [ ] D) $3 - x$

# Explicación
$\frac{x^2 - 9}{x - 3} = \frac{(x+3)(x-3)}{x-3} = x + 3$ (diferencia de cuadrados)
