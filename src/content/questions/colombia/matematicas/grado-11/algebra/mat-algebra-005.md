# Pregunta
Si $f(x) = 2x^2 - 3x + 1$, ¿cuál es el valor de $f(2)$?

# Opciones
- [ ] A) 1
- [x] B) 3
- [ ] C) 5
- [ ] D) 7

# Explicación
$f(2) = 2(2)^2 - 3(2) + 1 = 2(4) - 6 + 1 = 8 - 6 + 1 = 3$
