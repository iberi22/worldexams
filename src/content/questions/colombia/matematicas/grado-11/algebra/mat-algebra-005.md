# Pregunta
Si $f(x) = 2x^2 - 3x + 1$, ¿cuál es el valor de $f(2)$?

# Opciones
- [ ] A) 1
- [x] B) 3
- [ ] C) 5
- [ ] D) 7

# Explicación Pedagógica

**¿Por qué B es correcta?**
Para encontrar el valor de $f(2)$, sustituimos cada $x$ en la función $f(x)$ por el número 2. La operación es la siguiente: $f(2) = 2(2)^2 - 3(2) + 1$. Siguiendo el orden de las operaciones, primero calculamos la potencia: $2^2 = 4$. Luego, realizamos las multiplicaciones: $2(4) = 8$ y $-3(2) = -6$. Finalmente, sumamos y restamos los resultados: $8 - 6 + 1 = 2 + 1 = 3$.

**¿Por qué las otras opciones son incorrectas?**
- **A) 1:** Este error podría ocurrir si se calcula mal el orden de las operaciones, por ejemplo, restando 3 de 2 antes de multiplicar ($2(4) - 1 = 7$).
- **C) 5:** Este resultado se obtiene si se suma 6 en lugar de restarlo ($8+6+1=15$), o si se comete algún otro error de cálculo.
- **D) 7:** Este valor puede obtenerse si se olvida el signo negativo en el término -3x, calculando $8+6+1$ y cometiendo algún otro error.
