# Pregunta

Un parque cuadrado en Bogotá tiene un área de 64 metros cuadrados. ¿Cuál es la longitud de cada lado del parque?

# Opciones

- [ ] A) 6 metros
- [ ] B) 16 metros
- [x] C) 8 metros
- [ ] D) 32 metros

# Explicación

**Respuesta correcta:** C

Para un cuadrado, el área = lado². Si el área es 64 m², entonces:
- lado² = 64
- lado = √64 = 8 metros

**Por qué las otras opciones son incorrectas:**
- A) 6 metros daría un área de 36 m², no 64 m²
- B) 16 metros es el error de tomar la mitad del área en lugar de la raíz
- D) 32 metros es el error de tomar la mitad del área
