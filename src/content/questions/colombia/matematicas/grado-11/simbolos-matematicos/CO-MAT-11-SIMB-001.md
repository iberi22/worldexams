# Pregunta
¿Qué letra griega se utiliza para indicar una sumatoria?

# Opciones
- [ ] A) Delta (Δ)
- [ ] B) Alpha (α)
- [x] C) Sigma (Σ)
- [ ] D) Omega (Ω)

# Explicación
La letra griega Sigma mayúscula (Σ) se utiliza en matemáticas para representar sumatorias.

Ejemplo: Σ(i=1 hasta n) representa la suma de una secuencia.

Los distractores son otras letras griegas importantes:
- Delta (Δ): Representa cambio o diferencia
- Alpha (α): Primera letra, usada en ángulos y constantes
- Omega (Ω): Última letra, usada en física y resistencia
