# Pregunta
En cálculo, ¿qué letra griega se usa típicamente para representar un cambio infinitesimal?

# Opciones
- [x] A) Delta minúscula (δ)
- [ ] B) Epsilon (ε)
- [ ] C) Lambda (λ)
- [ ] D) Mu (μ)

# Explicación
La letra delta minúscula (δ) se utiliza en cálculo para representar cambios infinitesimales o límites.

Ejemplo: δx representa un cambio infinitesimal en x.

Los distractores:
- Epsilon (ε): Se usa para representar cantidades arbitrariamente pequeñas en límites
- Lambda (λ): Se usa en funciones lambda y longitudes de onda
- Mu (μ): Representa coeficientes de fricción o promedios
