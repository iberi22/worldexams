# Pregunta
¿Qué letra griega se utiliza comúnmente para representar la razón entre la circunferencia y el diámetro de un círculo?

# Opciones
- [x] A) Pi (π)
- [ ] B) Tau (τ)
- [ ] C) Phi (φ)
- [ ] D) Theta (θ)

# Explicación
La letra griega Pi (π) representa la constante matemática aproximadamente igual a 3.14159...

Fórmula: π = Circunferencia / Diámetro

Los distractores son otras letras griegas:
- Tau (τ): Algunas veces propuesto como 2π
- Phi (φ): Representa la razón áurea (≈1.618)
- Theta (θ): Comúnmente usado para ángulos
