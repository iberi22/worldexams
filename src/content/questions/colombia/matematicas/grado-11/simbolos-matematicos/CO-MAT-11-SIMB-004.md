# Pregunta
¿Qué letra griega se utiliza comúnmente para representar ángulos en geometría y trigonometría?

# Opciones
- [ ] A) Pi (π)
- [x] B) Theta (θ)
- [ ] C) Rho (ρ)
- [ ] D) Psi (ψ)

# Explicación
La letra griega Theta (θ) es la más comúnmente utilizada para representar ángulos en matemáticas.

Ejemplo: sen(θ), cos(θ), tan(θ)

Los distractores son otras letras griegas:
- Pi (π): Representa la constante 3.14159...
- Rho (ρ): Se usa para densidad o coordenadas polares
- Psi (ψ): Se usa en física cuántica para funciones de onda
