# Pregunta
¿Qué letra griega se utiliza para representar la razón áurea (aproximadamente 1.618)?

# Opciones
- [ ] A) Pi (π)
- [x] B) Phi (φ)
- [ ] C) Chi (χ)
- [ ] D) Eta (η)

# Explicación
La letra griega Phi (φ o Φ) representa la razón áurea, aproximadamente igual a 1.618033988...

Esta proporción aparece en la naturaleza, arte y arquitectura. Se calcula como: φ = (1 + √5) / 2

Los distractores son otras letras griegas importantes:
- Pi (π): Representa 3.14159... (relación circunferencia-diámetro)
- Chi (χ): Se usa en estadística (prueba chi-cuadrado)
- Eta (η): Se usa en física para eficiencia
