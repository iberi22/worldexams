# Pregunta
¿Qué letra griega se utiliza para representar el producto de una secuencia de números (análogo a Σ para sumas)?

# Opciones
- [ ] A) Sigma (Σ)
- [x] B) Pi mayúscula (Π)
- [ ] C) Gamma (Γ)
- [ ] D) Beta (β)

# Explicación
La letra Pi mayúscula (Π) se utiliza para representar el producto de una secuencia, similar a como Σ representa la suma.

Ejemplo: Π(i=1 hasta n) de i = 1 × 2 × 3 × ... × n = n!

Los distractores:
- Sigma (Σ): Se usa para sumatorias, no productos
- Gamma (Γ): Se usa en la función gamma (generalización del factorial)
- Beta (β): Se usa en estadística y física
