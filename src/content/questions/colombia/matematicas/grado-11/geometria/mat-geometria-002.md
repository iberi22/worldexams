# Pregunta
Un círculo tiene un radio de 7 cm. ¿Cuál es su área aproximada? (Use $\pi \approx 3.14$)

# Opciones
- [ ] A) 21.98 cm²
- [ ] B) 43.96 cm²
- [x] C) 153.86 cm²
- [ ] D) 307.72 cm²

# Explicación Pedagógica

**¿Por qué C es correcta?**
La fórmula para calcular el área de un círculo es $A = \pi r^2$, donde $r$ es el radio. Sustituyendo los valores dados, tenemos $A \approx 3.14 \times (7 \text{ cm})^2$. Primero, calculamos el cuadrado del radio: $7^2 = 49$. Luego, multiplicamos este valor por $\pi$: $A \approx 3.14 \times 49 \text{ cm}^2 = 153.86 \text{ cm}^2$.

**¿Por qué las otras opciones son incorrectas?**
- **A) 21.98 cm²:** Este valor se acerca más al resultado de $3.14 \times 7$, sin elevar el radio al cuadrado.
- **B) 43.96 cm²:** Este es el resultado de calcular la circunferencia del círculo ($2 \pi r = 2 \times 3.14 \times 7 = 43.96$), no el área. Es un error común confundir las dos fórmulas.
- **D) 307.72 cm²:** Este error puede surgir si se multiplica el área por 2, quizás confundiendo el $r^2$ con $2r$.
