# Pregunta
¿Cuál es el volumen de un cubo cuya arista mide 5 cm?

# Opciones
- [ ] A) 15 cm³
- [ ] B) 25 cm³
- [ ] C) 75 cm³
- [x] D) 125 cm³

# Explicación
Volumen del cubo = $arista^3 = 5^3 = 125$ cm³
