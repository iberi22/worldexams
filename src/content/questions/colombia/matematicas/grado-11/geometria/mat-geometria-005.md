# Pregunta
El perímetro de un rectángulo es 36 cm. Si su largo es el doble de su ancho, ¿cuál es el área del rectángulo?

# Opciones
- [ ] A) 36 cm²
- [ ] B) 54 cm²
- [x] C) 72 cm²
- [ ] D) 81 cm²

# Explicación
Si ancho = $x$, largo = $2x$. Perímetro: $2(x + 2x) = 36$ → $6x = 36$ → $x = 6$ cm. Área = $6 \times 12 = 72$ cm²
