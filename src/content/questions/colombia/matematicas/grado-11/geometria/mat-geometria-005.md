# Pregunta
El perímetro de un rectángulo es 36 cm. Si su largo es el doble de su ancho, ¿cuál es el área del rectángulo?

# Opciones
- [ ] A) 36 cm²
- [ ] B) 54 cm²
- [x] C) 72 cm²
- [ ] D) 81 cm²

# Explicación Pedagógica

**¿Por qué C es correcta?**
La fórmula del perímetro de un rectángulo es $P = 2(l + a)$, donde $l$ es el largo y $a$ es el ancho. Sabemos que $P = 36$ y que $l = 2a$. Sustituyendo estas condiciones en la fórmula: $36 = 2(2a + a)$, lo que se simplifica a $36 = 2(3a)$ o $36 = 6a$. Resolviendo para el ancho, $a = 36/6 = 6$ cm. Si el ancho es 6 cm, el largo es el doble, es decir, $l = 2 \times 6 = 12$ cm. El área de un rectángulo es $A = l \times a$. Por lo tanto, el área es $A = 12 \text{ cm} \times 6 \text{ cm} = 72 \text{ cm}^2$.

**¿Por qué las otras opciones son incorrectas?**
- **A) 36 cm²:** Este valor es el perímetro, no el área. Es un error común confundir los dos conceptos.
- **B) 54 cm²:** Este error puede surgir de un error de cálculo al resolver las ecuaciones para el largo y el ancho.
- **D) 81 cm²:** Este resultado (9x9) se obtendría si la figura fuera un cuadrado con el mismo perímetro ($36/4 = 9$), pero el problema especifica un rectángulo donde el largo es el doble del ancho.
