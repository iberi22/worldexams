# Pregunta
¿Cuál es el área de un triángulo con base de 10 cm y altura de 6 cm?

# Opciones
- [ ] A) 16 cm²
- [x] B) 30 cm²
- [ ] C) 60 cm²
- [ ] D) 36 cm²

# Explicación Pedagógica

**¿Por qué B es correcta?**
La fórmula para calcular el área de un triángulo es $\frac{base \times altura}{2}$. Sustituyendo los valores dados, tenemos $\frac{10 \text{ cm} \times 6 \text{ cm}}{2} = \frac{60 \text{ cm}^2}{2}$. Al realizar la división, obtenemos $30 \text{ cm}^2$.

**¿Por qué las otras opciones son incorrectas?**
- **A) 16 cm²:** Este error puede ocurrir si se suman la base y la altura ($10+6=16$) en lugar de multiplicarlas, confundiendo la fórmula del área con la del perímetro.
- **C) 60 cm²:** Este es el resultado de multiplicar la base por la altura, pero sin dividir por 2. Es un error común olvidar este último paso en la fórmula del área del triángulo.
- **D) 36 cm²:** Este error podría surgir de un error de cálculo en la multiplicación o de usar números incorrectos.
