# Pregunta

Si x + 3 = 7, ¿cuál es el valor de 2x + 5?

# Opciones

- [ ] A) 11
- [x] B) 13
- [ ] C) 15
- [ ] D) 9

# Explicación

**Respuesta correcta:** B

Primero resolvemos para x:
- x + 3 = 7
- x = 7 - 3
- x = 4

Ahora calculamos 2x + 5:
- 2(4) + 5 = 8 + 5 = 13

**Por qué las otras opciones son incorrectas:**
- A) 11 es el error de calcular 2x + 3 en lugar de 2x + 5
- C) 15 es el error de usar x = 5 en lugar de x = 4
- D) 9 es el error de calcular x + 5 en lugar de 2x + 5
