# Pregunta

¿Cuál es el resultado de la suma de fracciones: 1/2 + 1/4?

# Opciones

- [ ] A) 1/6
- [ ] B) 2/6
- [x] C) 3/4
- [ ] D) 1/3

# Explicación

**Respuesta correcta:** C

Para sumar fracciones con diferente denominador, primero encontramos el común denominador:
- 1/2 = 2/4
- 1/4 = 1/4
- 2/4 + 1/4 = 3/4

**Por qué las otras opciones son incorrectas:**
- A) 1/6 es un error al sumar incorrectamente los denominadores
- B) 2/6 es un error al calcular el común denominador o simplificar mal
- D) 1/3 es un error conceptual al manejar fracciones
