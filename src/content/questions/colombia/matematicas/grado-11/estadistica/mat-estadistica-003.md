# Pregunta
Se lanza un dado justo. ¿Cuál es la probabilidad de obtener un número par?

# Opciones
- [ ] A) 1/6
- [ ] B) 1/3
- [x] C) 1/2
- [ ] D) 2/3

# Explicación
