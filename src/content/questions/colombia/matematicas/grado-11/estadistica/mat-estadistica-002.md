# Pregunta
¿Cuál es la mediana del conjunto de datos: 2, 5, 8, 11, 14?

# Opciones
- [ ] A) 5
- [x] B) 8
- [ ] C) 11
- [ ] D) 7

# Explicación Pedagógica

**¿Por qué B es correcta?**
La mediana es el valor que ocupa la posición central en un conjunto de datos ordenado. El conjunto de datos proporcionado (2, 5, 8, 11, 14) ya está ordenado de menor a mayor. Como hay un número impar de datos (5 valores), la mediana es el valor que se encuentra exactamente en el medio. En este caso, el tercer valor es 8, por lo que la mediana es 8.

**¿Por qué las otras opciones son incorrectas?**
- **A) 5:** Este es el segundo valor del conjunto, no el central.
- **C) 11:** Este es el cuarto valor del conjunto, no el central.
- **D) 7:** Este valor no se encuentra en el conjunto de datos y podría ser el resultado de un cálculo incorrecto de la media.
