# Pregunta

En física y matemáticas, ¿cuál es el símbolo correcto para representar el desplazamiento?

# Opciones

- [x] A) $\Delta \vec{r}$ o $\vec{s}$
- [ ] B) $dr$
- [ ] C) $Dp$
- [ ] D) $r$

# Explicación

El desplazamiento se representa con:
- $\Delta \vec{r}$ (delta r vectorial): cambio en la posición
- $\vec{s}$ (s vectorial): vector desplazamiento

**Opciones incorrectas:**
- $dr$: diferencial de posición (cálculo infinitesimal)
- $Dp$: no es notación estándar
- $r$: posición, no desplazamiento (falta el $\Delta$)
