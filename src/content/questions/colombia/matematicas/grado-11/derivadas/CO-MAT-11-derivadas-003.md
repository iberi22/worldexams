# Pregunta

¿Cuál es la derivada de f(x) = x³?

# Opciones

- [ ] A) x²
- [x] B) 3x²
- [ ] C) x⁴
- [ ] D) 3x

# Explicación

**Respuesta correcta:** B

Usando la regla de potencias: si f(x) = xⁿ, entonces f'(x) = n·xⁿ⁻¹

Para f(x) = x³:
- n = 3
- f'(x) = 3·x³⁻¹ = 3x²

**Por qué las otras opciones son incorrectas:**
- A) x² es olvidar multiplicar por el exponente original (3)
- C) x⁴ es aumentar el exponente en lugar de reducirlo
- D) 3x es reducir el exponente demasiado (debería ser x²)
