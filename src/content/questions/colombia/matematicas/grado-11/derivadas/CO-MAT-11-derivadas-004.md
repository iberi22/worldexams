# Pregunta

¿Cuál es la derivada de f(x) = 4x² + 3x - 5?

# Opciones

- [ ] A) 4x + 3
- [x] B) 8x + 3
- [ ] C) 4x²
- [ ] D) 8x

# Explicación

**Respuesta correcta:** B

Derivamos término por término:
- Derivada de 4x² = 8x (usando la regla: d/dx(ax²) = 2ax)
- Derivada de 3x = 3
- Derivada de -5 = 0 (constante)
- Por lo tanto: f'(x) = 8x + 3

**Por qué las otras opciones son incorrectas:**
- A) 4x + 3 es olvidar que la derivada de x² es 2x, no x
- C) 4x² es no aplicar la regla de derivación correctamente
- D) 8x es olvidar sumar el término constante 3
