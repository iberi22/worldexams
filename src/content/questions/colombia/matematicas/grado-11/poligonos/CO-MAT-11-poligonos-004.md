# Pregunta

¿Cuál es la suma de los ángulos internos de un hexágono regular?

# Opciones

- [ ] A) 540°
- [ ] B) 600°
- [x] C) 720°
- [ ] D) 900°

# Explicación

**Respuesta correcta:** C

La fórmula para la suma de ángulos internos de un polígono es: (n - 2) × 180°, donde n es el número de lados.

Para un hexágono (n = 6):
- Suma = (6 - 2) × 180° = 4 × 180° = 720°

**Por qué las otras opciones son incorrectas:**
- A) 540° es la suma de ángulos de un pentágono (5 lados)
- B) 600° es un error al calcular incorrectamente la fórmula
- D) 900° es la suma de ángulos de un heptágono (7 lados)
