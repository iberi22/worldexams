# Pregunta

Un hexágono regular tiene un perímetro de 36 cm. Si el área del hexágono se puede aproximar usando la fórmula A ≈ 2.6 × L², donde L es la longitud de un lado, ¿cuál es el área aproximada?

# Opciones

- [ ] A) 78 cm²
- [x] B) 93.6 cm²
- [ ] C) 108 cm²
- [ ] D) 124.8 cm²

# Explicación

**Respuesta correcta:** B

Primero encontramos la longitud de cada lado:
- Perímetro = 6L = 36 cm
- L = 36/6 = 6 cm

Ahora calculamos el área:
- A ≈ 2.6 × L² = 2.6 × 6² = 2.6 × 36 = 93.6 cm²

**Por qué las otras opciones son incorrectas:**
- A) 78 cm² es el error de usar L = 5 en lugar de L = 6
- C) 108 cm² es el error de usar 3 × 36 sin la fórmula correcta
- D) 124.8 cm² es el error de calcular 2.6 × 48
