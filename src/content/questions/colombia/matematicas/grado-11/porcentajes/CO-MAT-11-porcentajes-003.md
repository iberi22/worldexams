# Pregunta

¿Cuánto es el 12.5% de 400?

# Opciones

- [ ] A) 40
- [ ] B) 45
- [x] C) 50
- [ ] D) 55

# Explicación

**Respuesta correcta:** C

Para calcular el 12.5% de 400:
- 12.5% = 12.5/100 = 0.125
- 0.125 × 400 = 50

Alternativamente: 12.5% es la mitad de 25%, y 25% de 400 es 100, entonces 100/2 = 50

**Por qué las otras opciones son incorrectas:**
- A) 40 es el 10% de 400
- B) 45 es aproximadamente el 11.25% de 400
- D) 55 es aproximadamente el 13.75% de 400
