# Pregunta

Un número de Mersenne se define como $M_n = 2^n - 1$. ¿Cuál de los siguientes es un número de Mersenne?

# Opciones

- [ ] A) 15 = $2^3 + 1$
- [x] B) 31 = $2^5 - 1$
- [ ] C) 33 = $2^5 + 1$
- [ ] D) 63 = $2^6 + 1$

# Explicación

Por definición, un número de Mersenne tiene la forma $2^n - 1$:
- $31 = 2^5 - 1 = 32 - 1$ ✓
- $15 = 2^4 - 1$, pero la opción dice $2^3 + 1$ ✗
- $33 = 2^5 + 1$ (forma incorrecta) ✗
- $63 = 2^6 - 1$, pero la opción dice $2^6 + 1$ ✗
