# Pregunta
Se tiene un círculo con radio r = 10 cm. Si se duplica el radio, ¿qué sucede con el área del círculo?

# Opciones
- [ ] A) El área se duplica.
- [ ] B) El área aumenta en 10 cm².
- [x] C) El área se cuadruplica.
- [ ] D) El área permanece igual.

# Explicación
Fórmula del área: A = πr².
Área inicial: A1 = π(10)² = 100π.
Nuevo radio: 20 cm.
Área final: A2 = π(20)² = 400π.
400π es 4 veces 100π.
Al duplicar el radio (x2), el área aumenta por el cuadrado del factor (2² = 4).

**Errores comunes:**
- A: Pensar linealmente (si radio x2, área x2).
