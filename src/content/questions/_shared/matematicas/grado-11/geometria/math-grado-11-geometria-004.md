# Pregunta
En un plano cartesiano, ¿cuál es la distancia entre los puntos A(1, 2) y B(4, 6)?

# Opciones
- [ ] A) 3 unidades
- [ ] B) 4 unidades
- [x] C) 5 unidades
- [ ] D) 7 unidades

# Explicación
Fórmula de distancia: d = √[(x2-x1)² + (y2-y1)²].
d = √[(4-1)² + (6-2)²]
d = √[3² + 4²]
d = √[9 + 16]
d = √25 = 5.

**Errores comunes:**
- A: Restar solo las x (4-1).
- D: Sumar las diferencias (3+4).
