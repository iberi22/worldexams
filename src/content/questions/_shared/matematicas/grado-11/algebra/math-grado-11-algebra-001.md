# Explicación
Para encontrar el vértice de una parábola dada por $f(x) = ax^2 + bx + c$, usamos la fórmula $x_v = -b / (2a)$.
Aquí, $a = 2$ y $b = -4$.
$x_v = -(-4) / (2 * 2) = 4 / 4 = 1$.
Para encontrar $y_v$, evaluamos $f(1)$:
$f(1) = 2(1)^2 - 4(1) + 1 = 2 - 4 + 1 = -1$.
Por lo tanto, el vértice es $(1, -1)$.
