# Pregunta

Dada la función $f(x) = x^2 - 4x + 3$, ¿cuáles son las coordenadas del vértice de la parábola?

# Opciones

- [ ] A) (2, -2)
- [ ] B) (-2, 15)
- [x] C) (2, -1)
- [ ] D) (-2, -1)

# Explicación

La respuesta correcta es **C) (2, -1)**.

Para una función cuadrática de la forma $ax^2 + bx + c$, la coordenada x del vértice es $h = -b/(2a)$.
Aquí, $a=1, b=-4$.
$x = -(-4)/(2*1) = 4/2 = 2$.

Para hallar y, evaluamos $f(2)$:
$f(2) = (2)^2 - 4(2) + 3 = 4 - 8 + 3 = -1$.
Vértice: $(2, -1)$.
